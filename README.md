# Elixir Medics - Medical Association Dashboard

A professional admin dashboard system for managing medical association operations, members, executives, resources, and announcements.

## Features

✅ **Admin Dashboard** - Professional sidebar navigation with real-time management
✅ **Member Management** - Register members and view all registrations
✅ **Executive Management** - Add, update, and delete executives with photo uploads
✅ **Feedback System** - Collect and review user feedback
✅ **Resource Management** - Upload PDF resources by subject
✅ **Announcements** - Create and publish announcements
✅ **Committees** - Manage committee information
✅ **Constitution** - Store and manage constitution text
✅ **Persistent Storage** - All data saved locally using localStorage

## Getting Started

1. Open `index.html` in a modern web browser
2. Click "Admin" to create an admin account
3. Access the admin dashboard to manage all features
4. Members can register using the "Sign Up" button
5. Members can submit feedback via the Contact form

## Admin Features

- Create admin account with email/password authentication
- View all registered members and remove members
- Add, update, and delete executives
- Upload photos directly from device or via URL
- Manage PDF resources for different subjects
- Create and publish announcements
- Configure committees
- Manage constitution document

## Technology Stack

- **HTML5** - Semantic markup
- **CSS3** - Responsive design with professional styling
- **JavaScript (ES6+)** - Client-side functionality
- **localStorage** - Data persistence

## Data Storage

All data is stored locally in the browser using localStorage. Data persists across sessions until cleared.
